## 目录结构
~~~
- assets   静态资源
- - anim      动画
- - color     颜色
- - drawable  样式
- - layout    布局
- - mipmap    图片
- - strings   文本
- - values    数据
- conf     配置文件(touch后编译生效,文件生成于src/common)
- docs     项目文档
- fonts    项目字体
- src      功能代码
- - common    常用函数
- - data      数据存储
- - function  功能函数
- - net       网络相关
- - protocol  通信协议
- - uart      串口相关
- - viewlibs  自定义页面
- - windows   页面函数
~~~
